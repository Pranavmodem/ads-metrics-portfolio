# Ad Metrics Design Portfolio

**Pranav Modem** | L5 Analytics Engineer Portfolio

## Overview

This is an interactive portfolio dashboard designed to showcase expertise in ad tech analytics, metrics design, and data visualization — directly aligned with the Netflix Advertising L5 Analytics Engineer role.

## Features

- **Attribution Model Comparison** — Interactive toggle between last-touch, first-touch, linear, and time-decay attribution models with D3.js Sankey diagram
- **Campaign Performance Dashboard** — Real-time KPI cards (CPM, CTR, ROAS, CPA, Fill Rate, Frequency) with 90-day trend analysis
- **Ad Supply & Decisioning Health** — D3 heatmap of hourly impressions by day of week, fill rate gauge, and latency distribution histogram
- **A/B Test Results Viewer** — Mock experiments with confidence intervals, p-values, and D3 bullet charts showing lift
- **Data Quality Scorecard** — Freshness, completeness, uniqueness, and accuracy metrics for key event streams
- **Architecture Diagram** — End-to-end data pipeline from ad servers through BigQuery, dbt, and dashboards
- **Netflix-Inspired Design** — Dark theme (#141414) with red accents (#E50914) and smooth animations

## Tech Stack

- **D3.js** — Interactive data visualizations (Sankey, heatmaps, bullet charts)
- **Chart.js** — Line, bar, and doughnut charts
- **Vanilla JavaScript** — Synthetic data generation and interactivity
- **HTML5 + CSS3** — Responsive, single-page application
